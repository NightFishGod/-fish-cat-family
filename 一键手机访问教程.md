# 📱 手机一键打开教程

## 方法一：ngrok（立即生效，零配置）

### 步骤
1. **下载 ngrok**：https://ngrok.com/download
2. **注册账户**：免费注册，获得认证 token
3. **运行命令**：
```bash
# 启动本地服务器
python -m http.server 8000 --bind 0.0.0.0 -d "D:\猫咪族谱网页版"

# 在另一个命令行（或终端）运行
ngrok http 8000
```
4. **获得链接**：ngrok 会生成类似 `https://abc-123.ngrok.io` 的链接
5. **手机扫码/复制链接**：打开浏览器即可

### 效果
- 链接永久有效（直到你关闭 ngrok）
- 支持 HTTPS
- 全球可访问
- 手机电脑都能开

---

## 方法二：GitHub Pages（永久免费）

### 步骤
1. 注册 GitHub
2. 创建仓库 `cat-family`
3. 上传 `D:\猫咪族谱网页版` 所有文件
4. 设置 → Pages → Source 选择 main 分支
5. 等待 1-2 分钟
6. 获得链接：`https://你的用户名.github.io/cat-family/`

### 优势
- 永久在线
- 完全免费
- 支持 HTTPS
- 可自定义域名

---

## 方法三：腾讯云/阿里云（国内访问快）

### 腾讯云 COS
1. 开通对象存储 COS
2. 创建存储桶，开启静态网站
3. 上传 `index.html`
4. 获得链接：`https://xxxx.cos.ap-shanghai.myqcloud.com`

### 阿里云 OSS
1. 开通对象存储 OSS
2. 创建存储空间，开启静态页面
3. 上传文件
4. 获得链接：`https://xxxx.oss-cn-hangzhou.aliyuncs.com`

## 方法五：局域网分享（无需互联网）

### 电脑端
```bash
cd "D:\猫咪族谱网页版"
python -m http.server 8000
```

### 手机端
1. 确保手机电脑同 Wi-Fi
2. 查看电脑 IP（cmd → `ipconfig` → IPv4 地址）
3. 手机浏览器输入：`http://电脑IP:8000`
   - 如：`http://192.168.1.100:8000`

## 🔧 遇到问题？

### 问题1：链接打不开
- 检查 ngrok 是否还在运行
- 确保本地服务器在运行
- 刷新页面

### 问题2：手机加载慢
- 图片太大，压缩一下
- 减少猫咪数量
- 使用更小的头像

### 问题3：想更新猫咪信息
- 修改 `index.html` 里的数据
- 重新上传到 GitHub
- 等待1分钟生效
