# 🐱 猫咪族谱网页版

一个记录14只猫咪的族谱网站，手机电脑都能随时访问。

## 🚀 快速开始

### 方法一：直接打开
**双击 `index.html`** → 浏览器打开即可使用

### 方法二：部署到GitHub Pages（推荐）
1. 双击运行 `一键上传到GitHub.bat`
2. 按照提示操作
3. 获得永久链接：`https://用户名.github.io/cat-family/`

### 方法三：局域网分享
1. 双击 `start.bat` → 选 1
2. 手机连同一Wi-Fi
3. 输入电脑IP地址：`http://192.168.xxx.xxx:8888`

## 📱 访问方式
| 方式 | 步骤 | 链接格式 |
|------|------|----------|
| **GitHub Pages** | 上传整个文件夹 | `https://用户名.github.io/cat-family/` |
| **局域网** | 同一Wi-Fi | `http://192.168.xxx.xxx:8888` |
| **U盘/网盘** | 复制文件夹分享 | 下载后双击 `index.html` |

## 🔧 自定义
### 修改猫咪信息
打开 `index.html`，找到 `const cats = [...]` 数组，修改名字、生日、简介等。

### 使用真实照片
1. 创建 `images/` 文件夹
2. 头像放 `images/avatars/`
3. 生活照放 `images/photos/`
4. 修改代码中的 `emoji` 为 `<img src="...">`
