# 🐱 猫咪族谱网站

一个简洁的猫咪族谱网站，记录14只猫咪的血缘关系。

## 🌐 访问网站
**在线地址：** `https://nightfishgod.github.io/-fish-cat-family/`

手机、电脑、平板均可直接访问。

## ✨ 功能特点
- 三代猫咪族谱树状展示
- 性别区分（♂蓝色，♀粉色）
- 点击头像查看详情和生活照片
- 简洁界面，无多余功能
- 主人更新，客人浏览模式

## 📝 如何更新猫咪信息
1. 打开 `index.html` 文件
2. 找到 `const cats = [...]` 数组（约第100行）
3. 修改猫咪的：名字、生日、性别、简介
4. 保存文件并推送到GitHub

## 🖼️ 如何添加真实照片
1. 创建 `images/avatars/` 文件夹存放头像
2. 创建 `images/photos/` 文件夹存放生活照
3. 在 `index.html` 中，将猫咪的 `emoji` 改为：
   ```html
   <img src="images/avatars/猫咪名字.jpg" alt="猫咪名字">
   ```

## 🚀 部署到GitHub
网站已部署到GitHub Pages，更新步骤：
1. 修改本地文件（`D:\猫咪族谱网页版\`）
2. 运行Git命令：
   ```bash
   cd D:\猫咪族谱网页版
   git add .
   git commit -m "更新猫咪信息"
   git push
   ```
3. 等待1-2分钟，刷新页面即可看到更新
